22 // A ReferenceParticle represents that particle which sits
23 // on the nominal orbit. It is responsible for maintaining
24 // the reference momentum and time (ct) for the bunch or
25 // map. ReferenceParticle cannot be instantiated, but is
26 // designed as a mixin for bunch or map-like classes.
