25 // Represents a Klystron, which can be connected to one or
26 // more RF cavities. A Klystron has a single Voltage and Phase
27 // which is then applied to the attached cavities in one of
28 // two modes:
29 //
30 // vector_sum:
31 // maintains any relative differences in
32 // local phase and voltage of the associated cavities, but
33 // scales them linearly to achieve the total correct vector
34 // sum.
35 //
36 // balanced:
37 // Sets each associated cavity to the identical phase and
38 // amplitude. If the request phase and ampitude of the klystron
39 // is set to the complex number z, then each of the n cavities
40 // is set to z/n.
