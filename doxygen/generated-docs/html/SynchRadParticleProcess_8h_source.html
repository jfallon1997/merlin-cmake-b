56 // Models the effects of synchrotron radiation in dipoles
57 // and (optionally) quadrupoles. The default behaviour is
58 // to include only the classical (average) effects. Calling
59 // GeneratPhotons(true) models the quantum effects due to
60 // random photon generation. The photon spectrum used can
61 // be changed by a call to SetPhotonGenerator(double
62 // (*)(double u)). The default spectrum (dipole radiation)
63 // has been provided by H. Burkhardt (CERN-LEP-Note 632).
64 //
65 // The number of equally spaced steps to take through a
66 // component can be specified (default = 1). The effect of
67 // the energy loss on the particles can be specified in two
68 // ways: first (default), only the dp/p are adjusted, and
69 // the reference energy (momentum) of the bunch is left
70 // unchanged; alternatively, the reference energy is adjusted to
71 // reflect the mean energy loss, and the dp/p then reflect
72 // the additional change with respect to the mean.
