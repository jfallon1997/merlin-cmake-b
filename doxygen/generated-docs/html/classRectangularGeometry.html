Return the three-dimensional transformation from the frame at s0 to the frame at s. s and s0 are in the geometry's s-frame, and must be within the geometry extents.

Parameters
[in]s0The location at which the transform should be evalutated from.
[in]sThe location at which the transform should be evalutated to.
Exceptions
Throwsa BeyondExtent exception if the requested s values are outside the geometry extent.
