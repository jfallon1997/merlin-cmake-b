30 // A charged particle bunch. Bunch fields a set of methods
31 // for accessing certain macroscopic quantities of a single
32 // bunch. An object of class Bunch represents an ensemble
33 // of like particle only. Functions for accessing the full
34 // six-dimensional phase space, as well as projection on a
35 // single phase plane are provided. Concrete classes should
36 // implement algorithm specific models of a bunch.
37
