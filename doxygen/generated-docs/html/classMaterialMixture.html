Detailed Description

A material mixture is a mixture of assorted materials, e.g. a metal alloy It contains a map with the component materials. Since it inherits from the base Material class, the same functions can be used in scattering, etc. Some properties will be the weighted mean of the component properties. Some will be discrete values where a material selection must be made at random. To support this the Material functions are virtual, and are overridden here.
