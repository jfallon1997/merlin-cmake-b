Detailed Description

A beam dynamics simulation. TrackingSimulation tracks a Bunch object through a specified Beamline (single pass). Tracking is performed by one or more BunchProcess objects, one of which is generally a transport process which is responsible for "tracking" the concrete Bunch implementation through the Beamline components. The initial Bunch is generated on each Run using the supplied BunchConstructor.
