Detailed Description

template<class P>
class BunchTracker< P >

A template class which is used to implement a bunch implementation specific TrackingSimulation. P must be a bunch specific transport process which has only a default constructor. It must also define P::bunch_type which specifies the type of bunch implementation to be tracked.
