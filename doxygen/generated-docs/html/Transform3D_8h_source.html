19 // Represents a 3-dimensional Euclidean transformation. A
20 // Euclidean transformation is characterised by a
21 // translation followed by a rotation. The translation
22 // vector and rotation can be thought of as the origin and
23 // orientation respectively of a set of mutually
24 // perpendicular axes. EuclideanTransform objects can act
25 // on Point3D and Vector3D objects, effectively
26 // transforming them into a new reference frame.
27 // EuclideanTransforms can also be composed using the
28 // multiplication operator *, i.e. T3 = T1*T2 (note that
29 // multiplication is not commutitive).
