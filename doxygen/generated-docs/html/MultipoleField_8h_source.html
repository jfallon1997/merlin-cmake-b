27 // A representation of a multipole expansion of a magnetic
28 // field having only a non-zero component of the magnetic
29 // vector potential in the z-direction. The general
30 // expression for the series is give as
31 //
32 //
33 // n
34 // By+i Bx = Sum B0 (bn-i an) (z/r0)
35 // i=0
36 //
37 // where z=(x+iy). B0 is the overall scale (in Tesla), and
38 // r0 is an aribitrary defining radius. For Multipole
39 // expansions representing ideal single multipoles (e.g.
40 // dipole, quadrupole etc), then B0 is set to the field
41 // strength at r0, and consequently |bn+ i an|=1.
42
