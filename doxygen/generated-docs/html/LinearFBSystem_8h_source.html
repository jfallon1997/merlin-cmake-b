30 // A simple linear feedback correction algorithm. On each
31 // application, the actuator channels (A) are incremented
32 // using the following linear equation:
33 //
34 // A = A-g*(Mi*(S-S0))
35 //
36 // where g is the gain, S are the current signal values and
37 // S0 are the desired signal values (set points). Mi is a
38 // speudo-inverse matrix of the response matrix M defined by
39 //
40 // S=M*A
41 //
42 // Mi is calculated using SVD.
