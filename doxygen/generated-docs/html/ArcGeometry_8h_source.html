21 // An ArcGeometry represents a constant radius curve in the
22 // local x-z plane. By convention, a positive curvature (h)
23 // defines a curve towards negative x. Transformations
24 // between two points (s1,s2) on an ArcGeometry are
25 // specified by a translation in the x-z plane of
26 // [(cos(phi)-1)/h, 0, sin(phi)/h], and a rotation about
27 // the y-axis of -phi (phi = h*(s2-s1)).
28 //
29 // An ArcGeometry can have an additional tilt, which is
30 // defined as a rotation about the local z-axis of the
31 // entrance plane by an angle theta, with an additional
32 // rotation about the exit plane z-axis by -theta.
33
