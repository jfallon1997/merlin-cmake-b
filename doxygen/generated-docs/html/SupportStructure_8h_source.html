28 // A SupportStructure represents a mechanical support
29 // system, on which accelerator components can be mounted.
30 // A SupportStructure is mounted to the ground by either
31 // one Support placed at the centre of enclosed geometry,
32 // or by two supports at the exit and entrance points.
33 //
34 // SupportStructure and its associated Support class are
35 // primarilly intended for application of ground motion and
36 // vibration.
